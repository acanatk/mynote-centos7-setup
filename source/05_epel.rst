**********************************************************************
EPELリポジトリセットアップ
**********************************************************************

EPELリポジトリのパッケージを使用するため、EPELリポジトリをセットアップします。

======================================================================
インストール
======================================================================

**epel-release** パッケージをインストールする。

.. code-block:: bash

    $ yum install epel-release
